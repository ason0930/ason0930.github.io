IBM  Web Marketing-Ruby

現在在IBM工作的 Ruby，也曾是微軟的實習生，不怕挑戰且勇於嘗試的她，分享了她多采多姿的求學過程與就業經歷，社團副社長、之咖啡店店員，資訊展業務員、活動企劃、 微軟實習生，透過不斷的努力與堅持，Ruby得到自己想做的工作！ 透過她的分享，讓所有參加者們感受到：人生，擁有無限的可能！

我現在在做什麼？

Ruby的現職是在 IBＭ 擔當 Web Marketing，「這個職位主要是管理 IBＭ 在全球每個據點當地的網站內容，也就是說當IBM的總公司或是分公司有什麼新資訊要發布時，每位 Web Marketing，就要將這些新資訊放到網站上。」 微軟實習生、IBＭ 的工作， Ruby豐富的經歷令人羨慕，她卻說：「在取得現在的成果之前，過去的累積是非常重要的。」她也曾經迷惘，不知道自己該做什麼，因此就什麼都去嘗試了，不論是咖啡店店員、活動企劃、展場業務…，在過程中，她了解到自己最想做的工作就是做業務。

了解自己是誰

「你要先了解自己是誰，才能知道自己想成為怎樣的人，這樣，才能夠開啟人生無限的可能！」「你們已經邁出第一步了!」Ruby 對著所有參加者說。正因為有想要破除的迷惘，有想要了解事物的熱情，所以大家才會在這，坐著聽講者們分享，也參與交流讓自己可以藉由別人的經驗，去吸收然後思考，該如何開啟自己人生的其他可能。


如何走到今天

Ruby 在大學時，曾受朋友邀約，去參加了未來領袖菁英養成學苑 (簡稱GYLA) 的課程，課程邀請到的業界名人讓她學習到許多，而同時，Ruby 做了一件很重要的事情：將身邊所有參與課程的學生的連絡資訊記錄下來，建立人脈網！在當中她找到隊友組隊參加 ATCC，又因ATCC，結交到更多有能力且滿懷抱負的夥伴，一起報名華碩校園CEO，雖然最後落選，卻讓 Ruby 有了動力報名擔任 Acer 的展場業務，去補強自己在硬體認知上的不足，最後因此被錄取成為微軟的實習生。Ruby發覺過去的經歷是會一圈又一圈的串一起的，最終會串成一個圍繞自己的大圓。

站在巨人的肩膀上看事情

進入微軟實習之後，讓 Ruby 學習到非常多大公司跟業界的潛規則，讓自己更能夠去應付在職場上的應對進退，而這些也有賴於前面過程的累積，同時也會讓自己深刻明白，當今天自己站在這個地方與位置，就不再是一個學生而已了，所有參與實習的學生，每個都是能力非常強的人，而 Ruby 的人生就在這裡產生了很大的改變，也遇到一位很照顧她的主管，那時她被問了一個問題：「妳想在這裡學到什麼？」，而 Ruby 回答：「我想學習如何做一個業務」，而這時也就訂下了日後進入IBM的契機，因為最好的業務都在IBM，讓 Ruby 從此開始為了進入IBM去努力。

為何要如此的努力

在大二的時候，Ruby 發現到那些平時在學校很風光的學長姐們，畢業後再回到校園來時，臉上的光彩都消失了，這讓 Ruby 警覺，如果不從現在開始磨練自己的心智與能力，未來也有可能會變成那樣。而在磨練的過程中，更讓 Ruby 了解到，如果沒有上好的學歷、特殊的家世、獨有的能力，只能透過努力再努力去彌補這些差距，讓自己離開學校時，能夠站得比別人更高更遠。實習期間，Ruby 就透過自己的努力，認識到許多有能力的人，還有主管，瞭解主管與員工之間的想法差異，最終讓自己在畢業之時，得以透過推薦進入IBM。


圈子對了，人生就對了

每個人的現在，都是過去累積來的。因此，如果想要一個好的未來，就也必須透過一個好的累積，在每個經歷與過程中，都放下一百分的心力去學習與努力，開放自己的心態，去認識所經歷的每個圈子中的各種人，累積各種不同的人脈，建立不同的視野，並替自己建立一個好的形象，讓別人可以足夠的信任你，並且推薦你去其他的公司做事。Ruby 勉勵參與活動的所有學生：「如果你還不知道自己想過怎樣的人生，那請一定要把握你的學生時光，盡可能地利用課餘的時間去接觸各種不同的事物，讓自己能夠結交到更多不一樣的人，不論是哪個圈子，都有值得學習與借鑒的人存在，去了解這些人，你才能夠知道那是不是你想要的。」
